# Optional: Routing — TanStack Router (file-based)

Use when the product needs **client-side routing** (SPA).

## Install

- `@tanstack/react-router`
- Dev: `@tanstack/router-plugin` (Vite plugin)

## Vite

```ts
import { tanstackRouter } from "@tanstack/router-plugin/vite";
// plugins: [tanstackRouter(), react(), ...]
```
